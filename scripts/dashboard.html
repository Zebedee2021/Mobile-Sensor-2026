传感器实时数据采集系统 — 使用说明

1. 系统架构

本系统通过 Sensor Logger App 采集手机传感器数据，经 HTTP POST 实时推送到本服务器，服务器同时保存 CSV 文件并通过 SSE 推送至本仪表盘。

2. 准备工作

  1. 手机安装 Sensor Logger (iOS / Android 均可, 免费)
  2. 电脑启动服务: python scripts/server.py (默认端口 8000)
  3. 浏览器打开 http://localhost:8000/dashboard

3. 局域网模式 (WiFi)

手机和电脑连接 同一个 WiFi 网络:

  1. 查看电脑局域网 IP (如 192.168.1.x)
  2. Sensor Logger → Settings → Enable HTTP Push
  3. Push URL 填入: http://<电脑IP>:8000/data
  4. 选择需要的传感器, 点击录制, 本页面即可看到实时波形
查看电脑 IP: Windows 打开 CMD 输入 ipconfig, 找到 "IPv4 地址"; Mac 打开终端输入 ifconfig en0

4. 5G/4G 远程模式 (ngrok 隧道)

手机使用移动网络, 和电脑 不在同一网络:

  1. 注册 ngrok.com 免费账号, 获取 authtoken
  2. 下载 ngrok, 配置: ngrok config add-authtoken <你的token>
  3. 启动隧道: ngrok http 8000
  4. 获得公网 URL (如 https://xxx.ngrok-free.dev)
  5. Sensor Logger Push URL 填入: https://xxx.ngrok-free.dev/data
  6. 仪表盘也可通过公网 URL 远程访问: https://xxx.ngrok-free.dev/dashboard
提示: ngrok 免费版有流量限制, 足够教学和测试使用。每次启动 ngrok 公网 URL 会变化。

5. Sensor Logger 设置建议

  • HTTP Push: 免费功能, 打开即可
  • 推荐传感器: Accelerometer, Gyroscope, Gravity, Orientation
  • 采样率: 默认 100Hz, 可在 App 内调整
  • Batch Period: 默认 1 秒推送一次 (免费版不可调)

6. 数据存储与分析

所有接收到的数据自动保存在服务器 data/ 目录, 每个录制 Session 生成一个 CSV 文件。

CSV 格式: time_ns, device, sensor, x, y, z, extra

后续可使用 python scripts/analyze_5g_data.py 对采集数据进行离线分析出图。

加速度计 (Accelerometer)

原理: MEMS 微机电系统, 通过测量微小质量块在加速时的位移来感知加速度, 含重力分量。
单位: m/s², 静止时合加速度 |a| ≈ 9.81 (地球重力加速度)。
校准 vs 未校准: 校准版已去除硬件零偏 (bias) 和比例因子误差; 未校准版为 MEMS 芯片原始输出。
典型应用: 计步器、跌倒检测、手势识别、屏幕旋转、车辆碰撞检测。

陀螺仪 (Gyroscope)

原理: 利用科里奥利力效应, 测量设备绕 X/Y/Z 轴的角速度。
单位: rad/s, 静止时各轴应接近 0, 快速旋转可达数 rad/s。
校准 vs 未校准: 差异通常很小, 现代 MEMS 陀螺仪出厂校准精度已很高。通过对比可验证零偏漂移。
典型应用: 姿态估计 (IMU)、光学防抖、导航 (惯性导航)、游戏手柄控制。

重力与方向 (Gravity & Orientation)

重力传感器: 软件虚拟传感器, 通过低通滤波从加速度计数据中分离重力分量。合向量 |G| 恒等于 9.81。三轴值反映设备在重力场中的朝向。
方向传感器: 融合加速度计 + 陀螺仪 + 磁力计 (Sensor Fusion), 输出欧拉角:
  • Yaw (偏航): 绕 Z 轴旋转, 即方位角/罗盘朝向, 范围 -180° ~ 180°
  • Pitch (俯仰): 绕 X 轴旋转, 手机前后倾斜
  • Roll (翻滚): 绕 Y 轴旋转, 手机左右倾斜
四元数: 同时输出四元数 (qw, qx, qy, qz), 范数始终为 1.0, 可验证融合算法精度。
