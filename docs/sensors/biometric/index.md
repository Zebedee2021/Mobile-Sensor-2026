# 生物识别类传感器

生物识别传感器利用人体独特的生理特征进行身份验证和健康监测。

---

## 本章概要

| 传感器 | 识别特征 | 核心技术 | 安全等级 |
|:-------|:---------|:---------|:---------|
| 指纹传感器 | 指纹纹路 | 电容/光学/超声波 | 高 (FAR < 1/50,000) |
| 面部识别 | 3D 面部几何 | 结构光 / ToF | 极高 (FAR < 1/1,000,000) |
| 心率传感器 | 脉搏波 | PPG (光电容积描记) | — |
| 血氧传感器 | 血红蛋白光吸收 | 双波长 PPG | — |

!!! note "FAR (误识率)"
    FAR (False Acceptance Rate) 是衡量生物识别安全性的关键指标,表示将非授权用户误认为授权用户的概率。
