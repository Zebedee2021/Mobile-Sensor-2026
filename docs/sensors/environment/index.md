# 环境类传感器

环境类传感器用于感知手机周围的物理环境参数,包括大气压力、光照强度、温度和湿度等。

---

## 本章概要

| 传感器 | 物理量 | 单位 | 核心原理 |
|:-------|:-------|:-----|:---------|
| 气压计 | 大气压力 | hPa (百帕) | MEMS 薄膜形变 → 电容/压阻变化 |
| 环境光传感器 | 光照强度 | lux | 光电二极管光电转换 |
| 温度传感器 | 温度 | °C | 热敏电阻 / PN 结温度特性 |
| 湿度传感器 | 相对湿度 | %RH | 电容式吸湿介质 |

---

## 在手机中的典型应用

| 应用场景 | 主要传感器 | 说明 |
|:---------|:----------|:-----|
| 海拔估算 | 气压计 | 大气压力与海拔的对应关系 |
| 楼层检测 | 气压计 | 检测层间气压差 (~0.12 hPa/层) |
| 自动亮度 | 环境光传感器 | 根据环境光自动调节屏幕亮度 |
| 相机 Flicker 检测 | 环境光传感器 | 检测 50/60 Hz 灯光频闪 |
| 电池热保护 | 温度传感器 | 监控电池与 SoC 温度 |
